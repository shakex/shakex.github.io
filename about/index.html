Project

Face Recognition experiment based on Fisherface, 2017 Code

This project is a coursework of Major Direction Practice (Intelligent Information Processing). Fisherface is a classic face recognition algorithm. It basically uses Principal Component Analysis (PCA) for dimensionality reduction and Linear Discriminant Analysis (LDA) for better classification. In this case, I used the ORL database of faces as the training data and running the algorithm code with Matlab.

Game: My Spirit, 2016 PPT Intro

My Spirit is an adventure/puzzle game using 2d hand-drawn art style. It tells the story about the legendary life of Jerry from the spirit world to the real world. My Spirit discusses the relationship between people’s body and their spirit. Players use keyboard or xbox one controller to control the characters. With the unique gameplay called ‘Out of Body’, players have the acess to switch the protagonist’s state between ‘body’ and ‘soul’ so that we can help Jerry overcome a lot of tricky problems. This project competed in the Imagine Cup 2016 National Finals and won the second prize.

Software & Harware: FarmKit, 2015 Intro

FarmKit is an open-source, software and hardware framework made specially for agricultural research. It consists of three parts: Intelligent Agricultural Control Terminal, Robot Surveillance and Control System and the Greenhouse Control System.
